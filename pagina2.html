Actualmente, se estima que hay más de 200 lenguajes de programación, es así que el índice TIOBE lenguajes de programación, nos brinda una clasificación o ranking para determinar el lenguaje más usuado en desarrollos de tipo público o privado. Este estudio se presenta de manera mensual y publica los 10 principales lenguajes de programación con mayor popularidad.

1. PYTHON

Python es un lenguaje de facil lectura y escritura ya que se ademeja al lenguaje humano. Una de las características más representativas es que es un lengujaje multiplataforma, e interpretado, es decir que no requiere compilación para ejecutar las aplicaciónes con código Python, ya que utiliza un interpretador al lenguaje a máquina. Actualmente cuenta con un ranking del 13.44%,

2. C

Es conocido como "Lenguaje de Programación de Sistemas". Es un lenguaje de alto nivel de tipo estructurado, facilita la escritura de código de manera compacta y sencilla. su origen, se enfocó en la implementación de Sistemas Operativos como UNIX.Actualmente cuenta con un ranking del 13.13%,

3. JAVA

Es uno de los lenguajes que se ha caracterizado por impulsar en gran forma el mundo digital. Por medio de este lenguaje, se pueden desarrollar diferentes aplicaciones y servicios. Actualmente cuenta con un ranking del 11,59%,

4. C++

Es un lenguaje de programaci[on de alto nivel, el cual puede bajar a lenguaje ASSEMBLER. Es muy usado para la programación de videojuegos. Actualmente cuenta con un ranking del 10,00%,

4. C#

Este lenguaje fue lanzado por Microsoft teniendo en cuenta lo mejor de C y C++ y es diseñado para el uso en .NET, que permite la creación de aplicaciones en forma sencilla. Actualmente cuenta con un ranking del 5,65%,
